\begin{abstract}
    Portfolio sorting is a common practice used to uncover systematic cross sectional relationships between expected returns and stock characteristics, where the null hypothesis is that the expected return over low and high percentiles is the same. In this paper, we study the effect of sorting errors on the test outcome. First, when the sorting variable is observable, a source of error arises due to the discrepancy between the order statistics and the true quantiles. Second, when the sorting variable is unobservable, the ordered statistics need to be estimated and two additional types of errors can occur: a contamination error (including in a portfolio stocks that should be excluded), and an information loss error (discarding from a portfolio stocks that should be included). For the estimated sorting variables, we suggest an adaptive trimming that ensures that tests for mean equality have the right asymptotic size and unit power
\end{abstract}
